# Stmsheild-PCB-design
sheild STM for motors control 

# Overview
an old project in 2024 based on a custom STM32F446RE shield PCB that was developed to support robot navigation by providing a dedicated interface for motor control and encoder feedback.

# Key Features
- STM32F446RE-based control board (shield format)
- Motor control interface for navigation system
- Support for 2 encoder inputs
- Designed for robotics motion control applications
- Compact integration for embedded robotic systems

  # Skills Demonstrated
- Embedded systems design
- PCB design (STM32-based architecture) (sch+PCB)
